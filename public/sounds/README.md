# Monkey Sound File

Place a monkey sound file here named: `monkey.mp3`

## Where to get it (free, royalty-free):

1. Go to: https://pixabay.com/sound-effects/search/monkey/
2. Find a short monkey chatter/cartoon sound (2-3 seconds)
3. Click Download → MP3
4. Rename the file to: monkey.mp3
5. Place it in this folder: public/sounds/monkey.mp3
6. Commit and push to GitHub

The game will automatically use this file instead of the synthesized sound.
